PROJECT: SnapPark — Smart Parking Management System

ROLE IN THE PROJECT:
I designed and implemented the JavaFX desktop kiosk application. I handled
the frontend development of the physical terminal by building FXML layouts,
binding the Java controllers, and creating the interactive graphical flow
for physical entry and exit points.

LEARNING FROM THE PROJECT:
I mastered JavaFX UI components, FXML styling, and the MVC design pattern
in the context of desktop applications. I discovered how to link visual
elements to backend services and map hardware-like interactions to a
graphical interface.

CHALLENGES FACED:
Designing a desktop UI that scales perfectly to fullscreen across different
monitor resolutions was a hurdle. Additionally, preventing UI freezes
during heavy background operations required careful implementation of
JavaFX Task and Platform.runLater() multithreading.
