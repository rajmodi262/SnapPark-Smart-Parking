PROJECT: SnapPark — Smart Parking Management System

ROLE IN THE PROJECT:
I designed and developed the mobile-first web interface using HTML, CSS,
and Vanilla JavaScript. My primary focus was implementing the premium
UI/UX features, integrating glassmorphism designs, and ensuring the
interface was completely responsive for mobile screens.

LEARNING FROM THE PROJECT:
I learned how to build robust, purely responsive web applications without
relying on heavy frontend frameworks. I deepened my understanding of
modern CSS aesthetics, vanilla DOM manipulation, and utilizing asynchronous
JavaScript (Fetch API) to seamlessly communicate with our Java backend.

CHALLENGES FACED:
Ensuring strict cross-browser compatibility and achieving a smooth, native
"app-like" experience purely on a mobile browser was challenging.
Additionally, integrating the QR-code-based flow into a frictionless
mobile UI required extensive manual state management.

  E06 — Live Parking Duration Timer      (LocalDateTime + JS timer)
