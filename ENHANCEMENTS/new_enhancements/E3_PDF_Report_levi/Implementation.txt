STEP 3: Create Report DAO (Database Layer)
─────────────────────────────────────────
File: ReportDAO.java (NEW)

Method: getMonthlyFrequency(int year, int month)

SQL Logic:
  SELECT v.license_plate, v.phone_number, v.vehicle_type,
         COUNT(ps.id) AS freq
  FROM parking_sessions ps
  JOIN vehicles v ON v.id = ps.vehicle_id
  WHERE ps.entry_time LIKE '2026-04%'
  GROUP BY v.id, v.license_plate, v.phone_number, v.vehicle_type
  ORDER BY freq DESC, v.license_plate ASC

How it works:
  1. JOINs parking_sessions with vehicles table
  2. Filters by month using LIKE on entry_time prefix
  3. GROUPs by vehicle to count sessions per vehicle
  4. ORDERs by frequency (highest first), then plate (alphabetical)


STEP 4: Create PDF Generator Service
────────────────────────────────────
File: PdfReportService.java (NEW)

Method: generateMonthlyFrequencyReport(int year, int month)
Returns: byte[] (PDF binary) or null on error

PDF Structure:
  1. HEADER — Dark background (#0D1525) with:
     - "SNAPPARK — Smart Parking Management" in white
     - Report title in cyan (#00C2FF)
     - Metadata: period, timestamp, vehicle count

  2. TABLE — 5 columns with:
     - Column headers on dark background
     - Alternating row colors (white / #F1F5F9)
     - Monospace font for vehicle numbers
     - Cyan font for frequency numbers

  3. TOTAL ROW — Green background showing sum of all visits

  4. FOOTER — Italic text explaining data source

  Empty data: Shows "No records found for April 2026" message


STEP 5: Add API Endpoint
────────────────────────
File: WebServer.java

Endpoint: GET /api/report/frequency?year=2026&month=4

Logic:
  1. Parse year and month from query parameters
  2. Validate: year 2000-2100, month 1-12
  3. Call pdfReportService.generateMonthlyFrequencyReport()
  4. Set Content-Type: application/pdf
  5. Set Content-Disposition: attachment; filename="..."
  6. Write PDF bytes to response body

Error handling:
  - Invalid params → 400 error
  - Generation failure → 500 error
  - NumberFormatException → 400 error
