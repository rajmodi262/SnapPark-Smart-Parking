STEP 1: Retrieve Previous Visit (DAO Layer)
───────────────────────────────────────────
File: SessionDAO.java

Added method: findLastCompletedByVehicle(int vehicleId)
- SQL: SELECT * FROM parking_sessions
       WHERE vehicle_id = ? AND active = 0
       AND exit_time IS NOT NULL
       ORDER BY exit_time DESC LIMIT 1
- Returns the MOST RECENT completed session for that vehicle
- Returns null if this is the vehicle's first visit

Why "active = 0 AND exit_time IS NOT NULL"?
  We only want sessions that are fully completed (paid & exited).
  Active sessions or sessions without exit time are excluded.


STEP 2: Calculate Discount Tier (Service Layer)
──────────────────────────────────────────────
File: BillingService.java

Added method: calculateLoyaltyDiscount(previousExitTime, currentEntryTime)
- Calculates Duration between previous exit and current entry
- Returns: 10.0 if ≤ 24 hours
           5.0 if ≤ 7 days
           2.0 if ≤ 30 days
           0.0 otherwise

Added method: getLoyaltyLabel(discountPercent)
- Returns human-readable text:
  "Returning within 24hrs" / "Returning within 7 days" / etc.

Added method: applyDiscount(amount, discountPercent)
- Math: amount - (amount × percent / 100)
- Rounds to 2 decimal places
